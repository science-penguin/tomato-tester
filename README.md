Tomato Tester is a web app tool for classifying tomato ripness, it runs entirly in browser and dousnt send any images to the cloud. I made this project for my SAE, a rigorous, Supervised Agricultural Experience. This project utilizes <a href="https://teachablemachine.withgoogle.com/">Google Teachable Machine</a>, a free, light weight, embeddable image classification model, to classify a tomato as either ripe or not ripe. I used a combination of 2 datasets for this project: TomatoD and LaboroTomato. Like both of the prior, this project is licensed under CC BY-NC-SA 4.0.
