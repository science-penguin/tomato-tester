Tomato Tester is a web app tool for classifying tomato ripness, it runs entirly in browser and dousn't send any images to the cloud. I made this project for my SAE, a rigorous, Supervised Agricultural Experience. To learn more about SAE’s, go <a href="https://www.ffa.org/educator_resources/sae-for-all-student-guide-2/">here</a>. This project utilizes <a href="https://teachablemachine.withgoogle.com/">Google Teachable Machine</a>, a free, light weight, embeddable image classification model, to classify a tomato as either ripe or not ripe. I used a combination of 2 datasets for this project: <a href="https://github.com/up2metric/tomatOD">TomatoD</a> and <a href="https://github.com/laboroai/LaboroTomato">LaboroTomato</a>.
